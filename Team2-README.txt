** 프로그램에 출력되는 안내사항에 따라 입력해주세요.
** 입력 시 대소문자를 정확하게 입력해주세요.
** 다른 안내사항이 없다면, 소문자로 입력해주세요.

1. 로그인
- 사용자의 아이디와 비밀번호를 입력합니다.
- 저장된 정보와 일치하지 않을 시, 로그인에 실패하여 프로그램이 종료됩니다.


2. 선호장르 설정
- 사용자가 기존에 설정해둔 선호장르를 변경할 수 있습니다.


3. 메인 기능
3-0. 선호 장르 변경
 - 사용자가 기존에 설정해둔 선호장르를 변경할 수 있습니다.

3-1. 창작물 검색
1) 필터 설정
 - 검색 전에 언어(language), 성인관람가능(isAdult), 매체(media), 현재진행여부(status)에 대해 필터를 설정할 수 있습니다.
 - 필터는 여러 개를 설정할 수 있으며, 원한다면 설정하지 않을 수 있습니다.
 - 필터 설정이 종료되면 입력어 검색화면으로 넘어갑니다.
2) 입력어 검색
 - 제목, 창작자, 키워드로 검색할 수 있습니다.
 - 검색 결과는 각 창작물에 대한 게시글들의 like 개수의 합이 높은 순서대로 제공됩니다.

3-2. 내 타임라인
 - 내가 팔로우한 사용자들의 게시글과 내 선호장르의 창작물에 대한 게시글을 보여줍니다.
   - 콘솔 프로그램의 시각적 편의를 위해 게시물은 10개만 보여줍니다. (Phase4에서 개선예정)
 - 게시글들은 작성한 시간 순서대로 정렬되어 제공됩니다.

3-3. 전체 게시판
3-3-1. 전체 게시글 보기
 - 전체 게시글을 작성한 시간 순서대로 정렬하여 보여줍니다.
   - 콘솔 프로그램의 시각적 편의를 위해 게시물은 10개만 보여줍니다. (Phase4에서 개선예정)
 - 좀 더 자세히 보고싶은 글의 글번호를 입력하면 해당 게시글과 함께 댓글까지 보여줍니다.
   - 글 번호 입력 시 '#'을 제외하고 숫자만 입력해주세요.
 - 선택한 게시글에 대해 댓글을 작성할 수 있습니다.
 - 댓글 내용을 입력하면 댓글이 등록됩니다.

3-3-2. 입력어로 게시글 검색
 - 글 제목, 글의 내용, 작성자 아이디에 입력어를 포함하는 모든 게시글을 시간 순서로 정렬하여 보여줍니다.
   - 콘솔 프로그램의 시각적 편의를 위해 게시물은 10개만 보여줍니다. (Phase4에서 개선예정)
 - 좀 더 자세히 보고싶은 글의 글번호를 입력하면 해당 게시글과 함께 댓글까지 보여줍니다.
   - 글 번호 입력 시 '#'을 제외하고 숫자만 입력해주세요.
 - 선택한 게시글에 대해 댓글을 작성할 수 있습니다.
 - 댓글 내용을 입력하면 댓글이 등록됩니다.

3-3-3. 게시글 작성하기
 - 글의 제목, 작품 식별번호, 글 공개여부, 글 내용 순으로 입력합니다.

3-4. 마이페이지
3-4-1. 비밀번호 변경
 - 변경 전에 기존 비밀번호를 입력하여 옳은 비밀번호인지 확인합니다.
 - 확인 후 변경하고자하는 새 비밀번호를 입력할 수 있습니다.

3-4-2. 팔로잉 목록
 - 목록에 있는 사용자를 선택하면 그 사용자의 게시글을 보여줍니다.
 - 게시글을 본 후, 원한다면 해당 사용자를 언팔로우 할 수 있습니다.
